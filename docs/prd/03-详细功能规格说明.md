# FundsBot PRD 03：详细功能规格说明

> 文档目标：提供研发可直接落地的功能规格（输入、输出、规则、边界）。

## 1. 模块 A：账本管理
### A-1 新建账本
- 输入：账本名称、币种（默认 CNY）
- 规则：名称唯一（用户维度）、长度 2-20
- 输出：账本 ID

### A-2 账本汇总
- 输出字段：总资产、当日预估盈亏、累计盈亏、持仓数
- 规则：仅统计“持仓状态=持有中”

## 2. 模块 B：持仓管理

### B-1 手动添加持仓
- 必填：基金代码、份额、持仓成本、所属账本
- 选填：平台、买入时间、备注
- 校验：
  - 份额 > 0
  - 成本 >= 0
  - 基金代码必须可匹配基金库

### B-2 增仓/减仓
- 增仓：更新份额和加权成本
- 减仓：更新剩余份额，已卖出部分进入交易记录
- 当份额=0：状态改为“已清仓”但保留历史记录

### B-3 OCR 导入
- 输入：截图（支持 PNG/JPG）
- 输出：导入草稿列表（字段 + 置信度）
- 规则：
  - 低于阈值字段标红
  - 必须用户确认后入库

### B-4 CSV 导入
- 模板字段：基金代码, 份额, 成本, 平台, 账本
- 校验：逐行报错，支持部分成功

## 3. 模块 C：实时估值与总览

### C-1 首页卡片
- 字段：总资产、当日预估盈亏、累计盈亏、收益率
- 刷新策略：交易时段 1-5 分钟刷新；非交易时段手动刷新

### C-2 持仓列表
- 字段：基金名称、代码、估值涨跌、预估净值、持仓收益
- 排序：默认按当日收益额降序
- 筛选：账本、平台、自选标签

### C-3 基金详情
- 展示：近 1 月/3 月/6 月净值曲线、持仓收益曲线
- 操作：编辑持仓、查看交易记录

## 4. 模块 D：净值提醒

### D-1 提醒触发
- 条件：组合内基金净值更新达到“可结算状态”
- 输出：推送标题 + 摘要（当日收益、涨跌贡献）

### D-2 阈值提醒
- 用户可配置：盈利阈值、亏损阈值
- 防骚扰：同类提醒每日最多 1 次

### D-3 免打扰
- 规则：在免打扰时段，仅保留站内提醒，不发系统推送

## 5. 模块 E：报表

### E-1 周报
- 指标：周收益额、收益率、波动区间、贡献 Top5
- 生成时间：每周日 20:00（可调整）

### E-2 月报
- 指标：月收益、累计收益趋势、平台对比、基金表现
- 生成时间：每月最后一天净值更新后

### E-3 分享
- 支持：长图分享（v1.0）
- 不支持：PDF（v1.1）

## 6. 权限与安全
- 仅当前用户可见个人账本
- 敏感字段（如账户号）不入日志
- 导入原图可配置“即用即删”

## 7. 错误码建议（示例）
- FND-1001：基金代码不存在
- FND-1002：持仓字段缺失
- FND-2001：行情服务超时
- FND-3001：推送发送失败

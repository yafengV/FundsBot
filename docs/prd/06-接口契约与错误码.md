# FundsBot PRD 06：接口契约与错误码（研发落地版）

> 目标：给前后端联调与测试提供可执行接口规范。默认前缀 `/api/v1`，鉴权方式 Bearer Token。

## 1. 通用约定
- Content-Type: `application/json`
- 时间字段：ISO8601（UTC 存储，前端按时区展示）
- 金额字段：分（int64）
- 收益率字段：万分比（int32），展示时 `/10000`
- 分页：`page`（>=1）, `pageSize`（1-100）

## 2. 核心实体
- Ledger（账本）
- Position（持仓）
- PositionTxn（持仓交易记录）
- FundQuote（估值行情）
- NavSnapshot（净值快照）
- ReportSnapshot（报表快照）
- NotifyRule（提醒规则）

## 3. 接口清单

## 3.1 账本
### POST /ledgers
创建账本
```json
{ "name": "主账户", "currency": "CNY" }
```
响应：
```json
{ "id": "ldg_001", "name": "主账户", "currency": "CNY", "createdAt": "2026-03-07T07:00:00Z" }
```
校验：
- name 长度 2-20，用户维度唯一

### GET /ledgers
列出账本 + 汇总
响应字段：`id,name,totalAssetCents,todayPnlCents,totalPnlCents,positionCount`

## 3.2 持仓
### POST /positions
创建持仓
```json
{
  "ledgerId":"ldg_001",
  "fundCode":"161725",
  "shares":"1234.56",
  "costNav":"1.2345",
  "platform":"alipay",
  "buyDate":"2026-03-01"
}
```
响应：`id,status,shares,costNav,marketValueCents,totalPnlCents`

### PATCH /positions/{id}
更新持仓（增仓/减仓/备注）
```json
{ "op":"increase", "shares":"100.00", "tradeNav":"1.2200", "tradeDate":"2026-03-07" }
```
`op` 枚举：`increase|decrease|edit`

### DELETE /positions/{id}
删除持仓（软删）
- 请求头：`X-Confirm-Token`

### GET /positions
查询持仓（筛选+排序）
参数：`ledgerId,platform,sortBy(todayPnl|totalPnl|changeRate),order`

## 3.3 导入
### POST /imports/csv/validate
上传 CSV（base64 或对象存储 key）并返回校验结果
响应：
```json
{
  "summary": {"total":100,"ok":92,"failed":8},
  "errors":[{"row":7,"code":"FND-1002","message":"shares missing"}]
}
```

### POST /imports/csv/commit
提交通过校验的批次，返回 `batchId`

### POST /imports/ocr/draft
上传图片，返回草稿字段+置信度

### POST /imports/ocr/confirm
用户确认草稿后入库

## 3.4 首页与估值
### GET /dashboard/overview
响应：

```json
{
  "totalAssetCents": 120034500,

  "totalPnlCents": 340500,

  "lastQuoteAt":"2026-03-07T06:22:00Z"

```

### GET /quotes/realtime
参数：`fundCodes=161725,000001`

## 3.5 净值与提醒
### GET /nav/daily-summary?date=2026-03-07
返回当日最终盈亏汇总

### PUT /notify/rules
```json
{
  "enableNavDone": true,
  "profitThresholdCents": 20000,
  "lossThresholdCents": -30000,
  "dndStart": "23:00",
  "dndEnd": "08:00"
}
```

## 3.6 报表
### GET /reports/weekly?week=2026-W10
### GET /reports/monthly?month=2026-03
返回快照对象（不可回写）

## 4. 错误码（统一）
- FND-1001 基金代码不存在
- FND-1002 参数缺失/格式错误
- FND-1003 业务校验失败（如减仓超过持有）
- FND-2001 行情服务超时
- FND-2002 净值数据缺失
- FND-3001 推送发送失败
- FND-9001 系统内部错误

错误响应：
```json
{
  "requestId":"req_xxx",
  "error": {"code":"FND-1002","message":"shares required","field":"shares"}
}
```

## 5. 幂等与一致性
- 写接口支持 `Idempotency-Key`
- 导入提交接口按 `batchChecksum` 去重
- 净值日结任务支持“重跑覆盖同日期快照”但保留版本号

## 6. 测试最小样例数据
- 3 个账本，20 条持仓，覆盖：盈利/亏损/清仓/异常基金代码

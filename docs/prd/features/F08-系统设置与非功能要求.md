# F08 功能PRD：系统设置与非功能要求

## 1. 功能目标
提供必要配置能力，并确保性能、安全、稳定性达到上线要求。

## 2. 功能范围
- 通知权限引导
- 免打扰配置
- 时区配置
- 错误提示与日志追踪

## 3. 非功能指标
- 首页 p95 < 500ms（接口）
- 首屏渲染 < 2s
- 核心可用性 >= 99.9%
- 敏感信息脱敏

## 4. 运维要求
- 告警：行情延迟、报表任务失败、推送失败
- 灰度：5% -> 20% -> 50% -> 100%
- 回滚：出现收益异常立即回滚

## 5. 测试用例（按功能）
- F08-TC-001 通知权限关闭时引导正确
- F08-TC-002 性能压测达标
- F08-TC-003 关键日志可追踪 requestId

## 6. 验收标准
- 发布阻断项全部通过
- Sev1/Sev2 缺陷为 0

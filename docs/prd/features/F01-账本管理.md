# F01 功能PRD：账本管理

## 1. 功能目标
支持用户创建多个账本并进行隔离管理，同时提供“全部账本汇总”能力。

## 2. 用户故事
- 作为多账户用户，我希望把不同资金来源分账本管理，避免混淆。

## 3. 功能范围
- 创建账本
- 编辑账本名称
- 查看账本列表与汇总
- 账本切换与全局汇总

## 4. 业务规则
1. 账本名称：2-20 字，用户维度唯一
2. 默认账本不可删除（v1.0）
3. 汇总视图统计所有 `holding` 持仓

## 5. 页面与交互
- 页面：账本管理页
- 关键交互：新建弹窗、切换账本下拉、汇总开关

## 6. 接口映射
- `POST /api/v1/ledgers`
- `GET /api/v1/ledgers`

## 7. 数据结构映射
- 表：`ledger`
- 关键字段：`id,user_id,name,currency,is_default`

## 8. 测试用例（按功能）
- F01-TC-001 创建账本成功
- F01-TC-002 重名失败
- F01-TC-003 汇总金额正确
- F01-TC-004 账本切换隔离正确

## 9. 验收标准
- 账本创建成功率 > 99%
- 切换后数据隔离 100% 正确

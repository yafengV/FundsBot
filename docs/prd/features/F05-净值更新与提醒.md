# F05 功能PRD：净值更新与提醒

## 1. 功能目标
在净值更新后及时通知用户“当日最终收益”，并支持阈值提醒。

## 2. 用户故事
- 作为用户，我希望在净值出齐后收到总结，不错过关键波动。

## 3. 功能范围
- 净值更新完成提醒
- 盈利/亏损阈值提醒
- 免打扰时间
- 通知点击跳转到当日总结

## 4. 业务规则
1. 同类阈值提醒每日最多 1 次
2. 免打扰时段仅站内提醒
3. 净值提醒触发窗口默认 20:30-23:30

## 5. 接口映射
- `PUT /api/v1/notify/rules`
- `GET /api/v1/nav/daily-summary`

## 6. 数据结构映射
- `notify_rule`, `fund_nav_daily`, `pnl_daily_snapshot`

## 7. 测试用例（按功能）
- F05-TC-001 净值更新后10分钟内提醒
- F05-TC-002 阈值防抖生效
- F05-TC-003 免打扰生效

## 8. 验收标准
- 提醒触达率 >= 98%
- 重复提醒率 < 1%

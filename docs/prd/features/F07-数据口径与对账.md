# F07 功能PRD：数据口径与对账

## 1. 功能目标
保证“看板、提醒、报表”三端口径一致，避免收益计算争议。

## 2. 功能范围
- 统一指标口径
- 日对账任务
- 异常报警
- 重算机制

## 3. 核心口径
- 总资产 = Σ(份额 × 最新估值净值)
- 当日预估盈亏：盘中估值口径
- 当日最终盈亏：净值结算口径
- 累计盈亏 = 当前市值 + 已实现收益 - 累计投入

## 4. 数据结构映射
- `fund_quote_realtime`, `fund_nav_daily`, `pnl_daily_snapshot`

## 5. 测试用例（按功能）
- F07-TC-001 固定样本对账通过
- F07-TC-002 边界值正确
- F07-TC-003 跨日切换口径正确

## 6. 验收标准
- 对账误差 <= 0.1%
- 异常发现与告警时延 <= 5 分钟

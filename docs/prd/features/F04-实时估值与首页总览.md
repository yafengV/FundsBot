# F04 功能PRD：实时估值与首页总览

## 1. 功能目标
为用户提供开屏可见的资产与收益总览，盘中支持估值刷新。

## 2. 用户故事
- 作为日常查看用户，我希望打开就看到今天赚亏和组合变化。

## 3. 功能范围
- 首页总览卡片
- 持仓列表涨跌显示
- 排序与筛选
- 基金详情趋势

## 4. 业务规则
1. 交易时段分钟级刷新
2. 非交易时段展示最近有效值
3. 行情延迟需显示“更新时间”

## 5. 接口映射
- `GET /api/v1/dashboard/overview`
- `GET /api/v1/quotes/realtime`
- `GET /api/v1/positions`

## 6. 数据结构映射
- `fund_quote_realtime`, `position`

## 7. 测试用例（按功能）
- F04-TC-001 总览值计算正确
- F04-TC-002 刷新后数据更新
- F04-TC-003 排序筛选正确
- F04-TC-004 延迟场景提示正确

## 8. 验收标准
- 首页首屏 <= 2s
- 关键指标计算误差 <= 0.1%

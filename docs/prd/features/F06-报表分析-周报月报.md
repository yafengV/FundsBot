# F06 功能PRD：报表分析（周报/月报）

## 1. 功能目标
通过周/月周期报表帮助用户复盘收益结构与风险暴露。

## 2. 用户故事
- 作为定投用户，我希望每周每月看到收益来源和拖累项。

## 3. 功能范围
- 周报生成与查看
- 月报生成与查看
- 账本维度切换
- 长图分享

## 4. 业务规则
1. 报表按周期落快照
2. 快照不可被后续行情覆盖
3. 切账本时使用同一口径

## 5. 接口映射
- `GET /api/v1/reports/weekly`
- `GET /api/v1/reports/monthly`

## 6. 数据结构映射
- `report_snapshot`, `pnl_daily_snapshot`

## 7. 测试用例（按功能）
- F06-TC-001 周报字段完整
- F06-TC-002 月报口径正确
- F06-TC-003 快照不可改写

## 8. 验收标准
- 周报生成成功率 >= 99%
- 报表读取 p95 < 500ms

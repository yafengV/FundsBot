# FundsBot PRD 08：研发任务拆解与 DoD（可排期版）

> 目标：研发团队可直接基于该文档分配任务与评审交付质量。

## 1. 后端任务拆解
### BE-01 账本服务
- 输出：账本 CRUD、汇总接口
- DoD：接口单测覆盖率 >= 80%，支持分页/筛选

### BE-02 持仓服务
- 输出：持仓 CRUD、增减仓结算逻辑、交易记录
- DoD：加权成本与清仓状态经用例验证

### BE-03 导入服务
- 输出：CSV 校验与提交、OCR 草稿确认入库
- DoD：部分失败可回报行级错误

### BE-04 行情与净值服务
- 输出：实时估值拉取、净值日结任务
- DoD：重试与降级策略可观测
- **tushare集成任务**:
  1. tushare客户端封装（支持测试/生产环境切换）
  2. 基金基础信息获取（fund_basic接口）
  3. 实时估值数据获取（fund_quote接口）
  4. 历史净值数据获取（fund_nav接口）
  5. 数据缓存机制实现
  6. 积分使用监控和告警
- **环境配置**:

  - 生产环境：token `d06bcb6694d4a1150c1b3f3bf5bfa7015b6a57e91d61fcec7d4260e0`，端点 `https://api.tushare.pro`

### BE-05 通知服务
- 输出：规则配置、净值提醒、阈值提醒
- DoD：同类提醒防抖与免打扰正确

### BE-06 报表服务
- 输出：周报月报生成快照
- DoD：快照可追溯、不可被覆盖

## 2. 前端任务拆解
### FE-01 首页总览
- 输出：总资产、当日预估、累计收益卡片
- DoD：首屏渲染 < 2s

### FE-02 持仓页/详情页
- 输出：列表、筛选、排序、详情图表
- DoD：关键操作可追踪埋点

### FE-03 导入流程
- 输出：CSV 上传结果页、OCR 草稿确认页
- DoD：错误提示可定位到字段/行

### FE-04 提醒与设置
- 输出：阈值设置、免打扰、通知权限引导
- DoD：配置变更实时生效

### FE-05 报表页
- 输出：周报/月报、账本切换、分享长图
- DoD：关键指标与后端对齐

## 3. 测试任务拆解
### QA-01 测试计划
- 输出：覆盖矩阵 + 资源安排

### QA-02 自动化
- 输出：核心接口自动化、核心 UI 冒烟

### QA-03 对账机制
- 输出：每日固定样本收益对账报告

## 4. 运维与可观测任务
### OPS-01 监控
- 接口成功率、延迟、任务成功率、通知送达率

### OPS-02 告警
- 行情延迟告警、净值任务失败告警、推送失败告警

### OPS-03 灰度与回滚
- 灰度开关、回滚脚本、版本切流策略

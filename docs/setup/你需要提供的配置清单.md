# 你需要提供的配置清单（其余已自动化）

下面这些是必须由你提供的外部配置，无法自动替代：

## 必需（业务可运行）
1. `QUOTE_API_KEY`（实时估值数据）
2. `NAV_API_KEY`（基金净值数据）

## 可选（按功能启用）
3. `OCR_API_KEY`（截图识别导入）
4. `FCM_SERVER_KEY` / `APNS_*`（移动端推送）
5. `SENTRY_DSN`（错误监控）

## 安全建议
- 所有 key 仅放在 `.env` 或 CI secrets，不要写入代码库
- 每个环境（dev/staging/prod）使用不同 key
- key 泄露后立即轮换
